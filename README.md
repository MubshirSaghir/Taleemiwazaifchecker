# Taleemi Wazaif Checker

This repository contains a simple, user-friendly tool designed to help users check their eligibility for the Taleemi Wazaif program. The Taleemi Wazaif program is a critical initiative aimed at providing educational stipends to deserving students.

## About the Project
This tool allows users to estimate their eligibility by inputting their age and class level. It features a clean, responsive interface built with HTML, CSS, and JavaScript, with built-in Urdu/English language toggling for maximum accessibility.

## 🔗 Official Resource
For more comprehensive guides, updates, and detailed information regarding the BISP and Ehsaas 8171 programs (including Taleemi Wazaif), please visit our official website:

**[Ehsaas Program 8171 Guides](https://8171guides.com)**

## Features
- **Instant Eligibility Check**: Quick verification based on age and class.
- **Bilingual Interface**: Seamless switching between Urdu and English.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop devices.
- **Clean UI**: Easy-to-use interface for all users.

## Setup Instructions
1. Clone this repository to your local machine.
2. Open `index.html` in your web browser.
3. Alternatively, deploy it using GitHub Pages for online access.
