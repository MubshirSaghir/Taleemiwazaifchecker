# BISP PMT Score Calculator

This repository contains a simple, user-friendly calculator designed to help users calculate their Proxy Means Test (PMT) score. The PMT score is a critical metric used to determine eligibility for various social protection programs.

## About the Project
This tool allows users to estimate their PMT score by evaluating specific socio-economic indicators. It features a clean, responsive interface built with HTML, CSS, and JavaScript.

## Features
- **Instant Calculation**: Quick PMT score estimation.
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices.
- **Clean UI**: Easy-to-use interface for all users.

## Setup Instructions
1. Clone this repository to your local machine.
2. Open `index.html` in your web browser.
3. Alternatively, deploy it using Netlify or GitHub Pages for online access.
